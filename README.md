PROYECTOS FULL STACK - JAVA/SPRING BOOT + HTML/JS/CSS
======================================================

DESCRIPCIÓN
-----------
Repositorio con proyectos full stack:
- Backend: Java + Spring Boot + MySQL + JWT
- Frontend: HTML5 + JavaScript + CSS

TECNOLOGÍAS
-----------
Backend:
- Java 17
- Spring Boot 3.x
- Maven
- MySQL 8.x
- JWT

Frontend:
- HTML5
- JavaScript
- CSS3
- Fetch API

REQUISITOS
----------
Para Backend:
- Java JDK 17+
- Maven
- MySQL 8.x

Para Frontend:
- Navegador web moderno
- Servidor web (opcional)
- Python Instalado

ESTRUCTURA DE PROYECTOS
-----------------------
proyecto-ejemplo/
├── backend/          (Spring Boot + Java)
├── frontend/         (HTML, JS, CSS)
└── README.md

INSTRUCCIONES DE EJECUCIÓN
--------------------------
1. Clonar repositorio:
git clone https://github.com/torrescampi/proyectos.git

2. Backend:
cd proyecto/backend
mvn spring-boot:run

3. Frontend:
cd proyecto/frontend
python -m http.server 8080

4. Abrir navegador:
http://localhost:8080
